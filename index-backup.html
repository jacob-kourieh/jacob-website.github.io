My Skills

In my role as a Frontend Developer, my expertise includes:

HTML, CSS, JavaScript, React, Angular, Angular Material, Redux, APIs, UX and Usability, Figma, Github, Dynamic web development, TypeScript, Node.js, MongoDB, Docker and Agile methods (Scrum).


In addition to these core skills, I have experience working with Material-UI, Mantine, and Tailwind CSS. MUI(Material-UI) is a popular React UI framework featuring a comprehensive set of material-design components for building rich user interfaces. Mantine is a modern React component library that provides a comprehensive set of components for building high-quality, responsive user interfaces. Tailwind CSS is a highly customizable, utility-first CSS framework for rapidly building custom designs and ensuring a consistent look and feel across a project.


I utilize my skills to create interactive and accessible websites while driving the development of larger projects. Whether working independently or collaborating within a team, my focus is on creating dynamic and engaging web experiences for users.

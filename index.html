My Skills and Experiences

In my role as a Frontend Developer, my expertise includes:

HTML, CSS, JavaScript, React, Redux, APIs, UX and Usability, Figma, Github, Dynamic web development, TypeScript, Node.js, MongoDB, and Agile methods (Scrum).


In addition to these core skills, I have experience working with Mantine and Tailwind CSS. Mantine is a modern React component library that provides a comprehensive set of components for building high-quality, responsive user interfaces. Tailwind CSS is a highly customizable, utility-first CSS framework for rapidly building custom designs and ensuring a consistent look and feel across a project.


I utilize my skills to create interactive and accessible websites while driving the development of larger projects. Whether working independently or collaborating within a team, my focus is on creating dynamic and engaging web experiences for users.
